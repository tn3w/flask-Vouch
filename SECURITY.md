# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in 𐌅𐌋𐌀𐌔𐌊-ᕓꝊ𐌵𐌂𐋅, please report it responsibly.

**Do not open a public issue.**

Instead, use [GitHub's private vulnerability reporting](https://github.com/tn3w/flask-Vouch/security) with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We aim to acknowledge reports within 48 hours and provide a fix or mitigation plan within 7 days.

## Supported versions

| Version | Supported |
| ------- | --------- |
| latest  | ✓         |

## Scope

Security issues in 𐌅𐌋𐌀𐌔𐌊-ᕓꝊ𐌵𐌂𐋅's core middleware, challenge generation, cookie signing, rate limiting, and framework integrations are in scope. Issues in third-party dependencies should be reported to their maintainers.
